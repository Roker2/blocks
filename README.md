Blocks
================
This is a simple text-based tetris clone for Linux.

Compiling
--------------
You will need ncurses to compile this. On Debian:

        apt-get install libncurses5-dev

And then

        make && make doc
        ./bin/blocks

Controls
-------------
* Move: Right, down, left
* Rotate: Up
* Drop: Spacebar
* Quit: q
